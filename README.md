Go Bricks is a collection of modular building blocks for Go. It provides reusable, foundational components that help you craft scalable systems with ease. Your foundation for clean, scalable Go.

## Modules

- [`pagination`](pagination/) — production-ready offset & cursor pagination (JSON:API / RFC 8288), opaque HMAC-signed cursors, and HATEOAS links.
